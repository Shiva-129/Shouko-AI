CRITICAL BUG FIXES (from code review)
--------------------------------------

1. Test Conftest Env Var Timing:
   - conftest.py was setting environment variables AFTER the Settings()
     singleton was already initialized at module level
   - Fix: Moved os.environ.setdefault() calls before all project imports
   - Now ENVIRONMENT=test is set before Settings() construction, so the
     validator skips the required secrets check

2. Sync boto3 in Async Methods:
   - storage_service.py was calling sync boto3 methods (put_object,
     get_object, etc.) directly inside async def methods
   - This blocks the asyncio event loop on network I/O
   - Fix: All boto3 calls wrapped in await asyncio.to_thread()

3. Dead Import:
   - Frontend artifact page imported useRef from React but never used it
   - Fix: Changed to useEffect, useState (useState was needed for retrying state)

4. Web Docker Healthcheck:
   - Used wget which is not available in node:20-alpine
   - Fix: Changed to node -e with inline http.get() request

5. .env.example Template Prefix:
   - File started with stray [TEMPLATE] marker
   - Fix: Removed via sed

6. Stale Config Key Reference:
   - billing_service.py referenced settings.STRIPE_API_KEY (old name)
   - Fix: Changed to settings.STRIPE_SECRET_KEY
